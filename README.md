# Experiência Prática 4 – SQL

Este repositório contém os scripts SQL desenvolvidos para a atividade da disciplina de Banco de Dados.  
O objetivo foi criar as tabelas do projeto, inserir dados e realizar consultas, atualizações e exclusões utilizando comandos SQL.

## Sobre o projeto
O mini-mundo utilizado é o de uma clínica chamada **Vida Saúde**, com as seguintes tabelas:
- Paciente
- Medico
- Especialidade
- Consulta

As tabelas possuem relacionamentos por meio de chaves primárias e estrangeiras.

## O que foi feito
- Criação das tabelas
- Inserts para povoar o banco
- Consultas com SELECT (incluindo JOIN)
- Comandos UPDATE
- Comandos DELETE

Para executar este projeto, basta abrir qualquer ferramenta SQL (MySQL Workbench, PGAdmin, Azure Data Studio, DBeaver, etc.), criar um banco de dados vazio e rodar os arquivos na seguinte ordem:

1️⃣ 01_criacao_tabelas.sql  
2️⃣ 02_inserts.sql  
3️⃣ 03_selects.sql  
4️⃣ 04_updates_deletes.sql  

Cada arquivo contém uma parte do projeto, facilitando a execução e a correção.

## Arquivo principal
Todo o código está no arquivo:
